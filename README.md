# DispLyrics

This is a chrome extension using which makes use of web scraping in beautiful soup and selenium.\
  Dom manupilations are done in javascript.\
  The main task is to fetch the songs that are being played on spotify.com and display lyrics on the spotify webpage if the user requires.\
  The popup window facilitate the users selection on the lyrics to be displayed on the webpage.\
  The python scripts and javascript scripts interact via json.\
  Using selenuim we make the search request on the lyrics webpage and display the results on the popup extention page on the chrome browser.\
  The user then selects the songs he wants to see the lyrics page and then the lyrics are displayed on the spotify webpage.
